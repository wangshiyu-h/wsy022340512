# 数据可视化仪表板

## 📊 项目介绍

基于matplotlib数据展示的现代化交互界面，采用金粉色渐变主题和卡片式布局的专业数据可视化项目。

## ✨ 主要特性

### 🎨 视觉设计
- **金粉色渐变背景** - 动态5色渐变效果
- **粉白色细闪点缀** - 60个飘落光点动画
- **卡片式布局** - 现代化响应式设计
- **平滑动画效果** - 所有交互都有流畅过渡

### 📈 图表类型
1. **折线图** - 数据趋势分析，支持悬停显示
2. **柱状图** - 数据分布情况，动态高度变化
3. **饼图/圆环图** - 数据占比分析，支持一键切换
4. **箱线图** - 数据分布分析，五数概括展示
5. **散点图** - 数据关联性，专业坐标系显示
6. **热力图** - 数据矩阵，96个数据单元可视化

### 🎯 交互功能
- **实时数据更新** - 支持手动和自动更新模式
- **饼图/圆环切换** - 点击按钮切换显示模式
- **悬停显示数值** - 所有图表支持鼠标悬停显示详细信息
- **动态数据生成** - 每次更新生成新的随机数据
- **统计卡片** - 5个可点击更新的统计指标

## 🎮 控制功能

- 🔄 **重新生成数据** - 一键刷新所有图表数据

- ▶️ **自动更新** - 开启自动数据更新
- ⏸️ **停止更新** - 停止自动更新
